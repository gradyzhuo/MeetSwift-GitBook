#Swift Opensource!


#### 此為TCCM社群之分享課程，採一系列教學方式，分享作者於Swift開發一年過程的經驗談，原本課程是透過**Playground**剖析Swift 寫法的獨特之處，並即時帶入範例的運作，以期待達成所學即所得的效果。
#### 不過為避免Playground只能在Xcode中運行之困難，現在也一併發佈在Gitbook，給需要的朋友一個參考。
